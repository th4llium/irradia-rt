# Irradia RT

**Irradia RT** is a high-performance, path-traced shader built specifically for the Minecraft Bedrock RTX pipeline. This project is designed from the ground up to deliver a perfect balance between cutting-edge visual realism and seamless gameplay.



## Features

*   **Physically Based Atmosphere:** Experience a fully dynamic and physically accurate atmospheric rendering system.
*   **Advanced Global Illumination:** Enjoy sharper, more accurate, and physically based Global Illumination (GI) for highly realistic lighting.
*   **Crystal Clear Reflections:** Features significantly sharper and cleaner reflections compared to standard implementations.



## System Compatibility

Irradia RT supports a variety of GPU vendors, provided they meet the hardware requirements:

*   **NVIDIA:** All RTX series graphics cards are fully supported.
*   **AMD:** Supported, provided the GPU features Hardware Accelerated Ray Tracing.
*   **Intel:** Supported, provided the GPU features Hardware Accelerated Ray Tracing.



## Installation Guide

You can install Irradia RT using a method:

### Method 1: Manual Material Replacement
*   Irradia RT can be installed by manually replacing the game's vanilla materials with the custom Irradia RT materials.
*   These materials are available for download in the [Releases](https://github.com/th4llium/Irradia-RT/releases/) section.
*   **Warning:** Please ensure you back up your vanilla materials before making any modifications.
